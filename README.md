# 外卖冲冲冲 - 微信小游戏

一款创新的箭头解谜类小游戏，将传统箭头解谜玩法与外卖配送题材结合。

## 项目简介

**游戏名称**：外卖冲冲冲
**游戏类型**：路径解谜类休闲游戏
**目标平台**：微信小游戏、抖音小游戏
**变现模式**：IAA（广告变现）
**开发模式**：个人开发，遵循SDD开发流程

## 核心玩法

玩家需要帮助外卖骑手规划出发顺序，让所有骑手避开障碍、按时送达订单。

- 每个骑手只能沿固定方向直线移动
- 碰到障碍会延迟或弹回
- 需要合理规划出发顺序，让所有骑手成功送达

## 差异化特色

1. **题材创新**：外卖配送主题，全民共鸣
2. **角色包装**：有表情、有台词的外卖骑手
3. **特殊机制**：红绿灯、电梯、VIP订单等
4. **传播性强**：搞怪台词适合短视频传播
