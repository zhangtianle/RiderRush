## 十、AI生成建议

### 10.1 Midjourney/DALL-E 通用提示词模板

```
Q版游戏角色，外卖骑手，简笔画风格，可爱，扁平化设计，
俯视图，游戏素材，高质量，无背景，透明PNG，
[具体描述]，[尺寸]
```

### 10.2 批量生成建议

1. **优先生成P0资源**：骑手图、阻碍图、UI按钮
2. **统一风格**：全部使用Q版简笔画风格，保持一致性
3. **透明背景**：确保PNG格式，便于游戏使用
4. **尺寸一致**：按指定尺寸生成，减少后期调整
