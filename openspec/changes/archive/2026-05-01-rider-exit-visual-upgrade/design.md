### renderExits() 改动

当前 renderExits() 只有一种 EXIT_PALETTE。需要改为：

1. 遍历 exits 时，查找哪些 rider 的 targetExit 指向该 exit
2. 根据 rider 类型选择对应的 EXIT_PALETTES
3. 如果多个不同类型的骑手指向同一 exit，使用第一个骑手的类型（或用默认色）
4. 脉冲光圈颜色也跟随调色板变化

```typescript
// 伪代码
private renderExits(level: Level): void {
  level.exits.forEach(exit => {
    // 找到指向该出口的骑手
    const assignedRiders = level.riders.filter(r => r.targetExit === exit.id);
    const riderType = assignedRiders[0]?.type || RiderType.NORMAL;
    const palette = EXIT_PALETTES[riderType] || EXIT_PALETTES.NORMAL;

  });
}
```
