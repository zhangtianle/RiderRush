## Spec: rider-sprite-detail

骑手像素精灵增加外卖箱、眼睛高光、头盔反光条，提升职业辨识度和生动感。

### 需求

1. RIDER_SPRITE 行3 眼睛像素从黑色(6)改为高光白(8)，形成"眼睛里有光"的效果
2. RIDER_SPRITE 行10 中间 4 格从夹克色(3)改为外卖箱色(9)，形成背箱子的视觉
3. 每种骑手调色板新增 index 8（高光色）和 index 9（外卖箱色）
4. CRASHED 状态的高光和箱子色为灰色系，保持整体灰暗感
5. 改动最小化，不改变精灵尺寸（保持 12×14），不影响布局参数

### 验收标准

- [ ] 骑手眼睛有白色高光点，看起来更生动
- [ ] 骑手背上能看到外卖箱色块，一眼认出是外卖骑手
- [ ] 4 种骑手类型（NORMAL/VIP/URGENT/CRASHED）各有合适的高光和箱子颜色
- [ ] 不影响现有碰撞检测、路径绘制、状态边框等逻辑
