# 计时修复测试指南

## 访问地址
http://localhost:8080

## 测试项目

### 1. 星级评分测试
- 选择一个有时间限制的关卡
- 快速完成（<30秒）→ 应获得3星
- 中等速度完成（30-60秒）→ 应获得2星
- 慢速完成（>60秒）→ 应获得1星

### 2. 浏览器后台切换测试
- 开始游戏
- 切换到其他标签页（等待5-10秒）
- 切回游戏标签页
- 验证：游戏状态应正常，无骑手瞬移或倒计时异常

### 3. 连击系统测试
- 快速连续送达2个骑手（2秒内）
- 验证：应触发连击效果

### 4. 复活功能测试
- 故意失败一个关卡
- 点击"分享复活"
- 验证：复活后时间应重置，游戏可正常继续

## 预期结果
- ✅ 星级评分基于实际完成时间
- ✅ 浏览器后台切换后游戏状态稳定
- ✅ 连击系统使用统一的时间基准
- ✅ 复活功能正常工作
